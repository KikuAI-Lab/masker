# Demo Page

Interactive demo for the Masker API.

## Files

| File | Description |
|------|-------------|
| `index.html` | Landing page with live demo |
| `styles.css` | Styles (dark theme, animations) |
| `script.js` | Interactive demo functionality |

## Usage

### Local Development

```bash
# Serve locally
python -m http.server 3000 -d demo/
# Open http://localhost:3000
```

### Deployment

Host on any static hosting:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Features

- Interactive PII redaction demo
- Animated landing page
- Dark cyberpunk theme
- Responsive design
