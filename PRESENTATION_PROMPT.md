# Промпт для создания презентации Masker

## Контекст проекта

**Masker** — это privacy-first HTTP API для обнаружения и удаления персональных данных (PII) из текста и JSON перед отправкой в LLM (ChatGPT, Claude, Gemini и др.).

### Ключевые характеристики:
- **Технологии**: FastAPI, Python 3.11, spaCy (NER), regex patterns
- **Производительность**: среднее время обработки 15-25ms
- **Архитектура**: stateless, без хранения данных, без логирования контента
- **Поддержка**: текст и JSON (рекурсивная обработка)
- **PII типы**: EMAIL, PHONE, CARD, PERSON (имена через NER)
- **Языки**: английский (en), русский (ru)
- **Режимы маскировки**: `mask` (***), `placeholder` (<EMAIL>, <PERSON>)
- **Монетизация**: RapidAPI интеграция, тарифы Free/Basic ($9)/Pro ($29)/Enterprise
- **Use cases**: очистка данных перед LLM, анонимизация тикетов поддержки, обработка форм, санитизация логов

## Технические детали для включения

### Примеры кода (Python):
```python
import requests
from openai import OpenAI

# Redact PII
response = requests.post(
    "https://masker.kikuai.dev/v1/redact",
    json={"text": "Contact John Doe at john@example.com", "mode": "placeholder"}
)
safe_text = response.json()["redacted_text"]

# Send to LLM
client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": safe_text}]
)
```

### Примеры кода (JavaScript):
```javascript
const response = await fetch('https://masker.kikuai.dev/v1/redact', {
```
